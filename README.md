# 💼 Job Portal Web Application

A multi-page Job Portal website where users can register either as a **Job Seeker** or a **Job Provider (Recruiter)**.

This project is built using HTML and CSS and demonstrates user interface design, multi-page navigation, and structured layout for a real-world web application.

---

## 🚀 Features

### 👨‍💼 Job Seeker
- Register / Login as Job Seeker
- Browse available jobs
- View job details
- Apply for jobs

### 🏢 Job Provider (Recruiter)
- Register / Login as Provider
- Post new job listings
- Manage posted jobs
- View applicants (UI based)

---
